# Overview  
This is Backend Profjects for application which helps users to make schedule for playing with their friends.

# Core Features  
In frontend, Users can make or join in some room which their friends can enter too.
The Backend should store user and room room data. Each users have their preference about food, play contents such as going beautiful cafe, board game cafe, watching movie etc.
This backend should have login, making editing, entering room, setting their preference and update their preference automatically by user data.
It also recommend best play contents schedule such as PM 1:00 restaurant, PM 2:00 cafe, etc, using the preference of members who enjoyed in same room.
